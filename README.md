# RUANG KARYA

Ruang karya adalah platform online untuk murid memposting karya mereka sebagai portfolio.
Siswa dapat mengupload karya mereka dan melihat karya orang lain.

## Fitur

ADMIN/GURU
- Membuat CRUD user (siswa, guru, admin)
- Mengkonfigurasi Web.
- Mengelola karya siswa.

SISWA
- Mengelola karya mereka.
- Mengelola profil mereka.

## Workflow

1. Guru membuat user siswa.
2. Siswa akan melakukan login.
3. Siswa akan mengisi data diri.
4. Siswa akan mengisi karya mereka.
5. Siswa akan melihat karya orang lain.
6. Siswa akan melihat profil orang lain.

## Teknologi

- Laravel
- Inertia (React)
- Tailwind CSS (Shadcn UI)
- Vite
- MySQL
- Sail
